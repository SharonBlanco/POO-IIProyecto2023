# TriviaQuirk

Proyecto realizado para el curso de Programación Orientada a Objetos.

Es un juego de trivia con interfaz gráfica donde uno o varios jugadores responden preguntas de distintas categorías.

---

## ¿Qué hace?

- Permite jugar en modo 1 jugador o multijugador
- Se pueden elegir categorías (cine, países, cultura general, etc.)
- Cada jugador responde preguntas por turnos
- Tiene temporizador por pregunta
- Al final muestra los puntajes y el ganador

---

## Tecnologías

- Java
- Swing
- JSON (para las preguntas)
